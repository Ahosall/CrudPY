# CrudPY

Um simples crud com Python, PySimpleGUI, e SQLite3.

## Ativação
Antes de tudo deveremos instalar as dependências
```bash
$ pip install -r requirements.txt
```
ou
```bash
$ python -m pip install -r requirements.txt
```
Para iniciar o projeto:
```bash
$ python app.py
```
